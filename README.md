# 퀵링크

`퀵링크`는 우측 사이드 패널에서 그룹별 바로가기 링크를 추가, 삭제, 정리할 수 있게 만들 예정인 Chrome 확장 프로그램이다.

현재는 기능 구현 전 단계이며, 아래 메타데이터와 기본 골격만 정리된 상태다.

- 프로젝트 이름: `퀵링크`
- 기본 UI: React 기반 우측 사이드패널
- 남겨둔 동작: 확장 프로그램 아이콘 클릭 시 사이드패널 열기/닫기

## 예정 기능

- 그룹별 바로가기 링크 관리
- 링크 추가 및 삭제
- 사이드패널 중심의 빠른 접근 UI

## 구조

- `src/sidepanel`: 퀵링크 패널 UI 골격
- `src/background/index.js`: 액션 아이콘 클릭 시 사이드패널 토글
- `public/manifest.json`: 확장 프로그램 메타데이터
